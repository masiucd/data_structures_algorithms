## Problem

Write a function that checks if a string is a palindrome

**Example 1:**

```
Input: "apa"

Output: true
```

**Example 2:**

```
Input: "racecar"
Output: "true"
```

**Example 3:**

```
Input: "apple"
Output: "false"
```

## Solution

```javascript title="Using reverse" showLineNumbers
function isPalindrome(str) {
  let left = 0;
  let right = str.length - 1;

  while (left < right) {
    if (str[left] !== str[right]) {
      return false;
    }

    left++;
    right--;
  }

  return true;
}
```
