# is palindrome

is_palindrome checks if a string is a palindrome.
for example "abba" is a palindrome, but "abab" is not.

## Example

```rs
assert_eq!(is_palindrome("abba"), true);
assert_eq!(is_palindrome("abab"), false);
```

## Complexity

- Time complexity: `O(n)`
- Space complexity: `O(1)`
