# Insert BST

## Description

Write a function that inserts a value into a Binary Search Tree.

## Example

```go
insert(5)
insert(3)
insert(7)
insert(2)

// Tree:
//       5
//     /   \
//    3     7
//   /
//  2
```

## Constraints

- **Time**: `O(log n)` average and `O(n)` worst case
- **Space**: `O(1)`
- Assume this BST only has `insert` and `contains` methods
